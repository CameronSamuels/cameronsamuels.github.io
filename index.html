Coming Soon

Memerator

Memerator is a meme generator built in JavaScript by 2 others and I. It allows you to upload an image, and add top and bottom text to it. You can choose the font, color, and rotation for the image. You can then download the meme. It is a fully working prototype, but will be improved.

WebAPP

WebAPP is going to be an iOS app for turning websites into native-like apps. You can input any URL in the settings app and then open WebAPP to use that website as an actual app. There will be more developer options to come and the ability to export the Xcode project.
