Fidget Spinner

"Fidget Spinner - the best one" is a digital version of a popular fad, the Fidget Spinner. Furthermore, this was my most popular app in the timeframe it was popular. It had over 300,000 installs on the iOS app store, and over 100,000 downloads on the Google Play Store. It was the top 35 app on iOS, and it had over 300 reviews overall.

Games

bClick

bClick was the first app I ever made. I made it when I was 9 years old. It is a clicking game inspired by Cookie Clicker. It is like my test dummy app that has made me get better at my coding skills through the years.

Tools

Dice Thrower

Dice Thrower is a virtual dice app for Chrome, Android and the web. I will soon publish this to iOS, Firefox, and possibly more.

Developer Tools

KeyCODE

KeyCODE is an web app for finding out JavaScript key codes. You can type any keyboard key to get the code value for it to use in your code.

PolyGEN

PolyGEN is a JavaScript library I made when I needed to generate shapes for Ludum Dare 35. It lets you easily generate a shape on an HTML5 canvas with any amount of sides.

Entertainment

WordMAKER

WordMAKER is an app that generates random words and pronounces them automatically. It has a UI and name inspired from KeyCODE. It is available for the web, Chrome, and Android. Soon to be published for iOS.

Coming Soon

Memerator

Memerator is a meme generator for any device built in JavaScript. It allows you to upload an image and add top and bottom text to it. You can choose the font, color, and rotation for the image. You can then download the meme. It has a beautiful material design and user-friendly color selecting. It is a fully working prototype, but will be improved. I am currently working with 2 people to finish this by July, but it may be finished at a different time.

WebAPP

WebAPP is going to be an iOS app for turning websites into native-like apps. You can input any URL in the settings app and then open WebAPP to use that website as an actual app. You can disable cache unlike non-native web apps. More features will probably come.
